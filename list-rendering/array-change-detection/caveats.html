Following changes cannot be detected by Vue.js:

  • When you directly set an item with the index, e.g.: caveatsModel.items[0] = {}
  • When you modify the length of the Array, e.g.: caveatsModel.items.length = 0

You should try this instead:
// same as `caveatsModel.items[0] = ...` but triggers view update
caveatsModel.items.$set(0, { message: 'Changed!'})

You can also use the remove() function
caveatsModel.items.$remove({ message: 'foo' })
This needs to be the same object instance.
